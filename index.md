## 关于这门课程

> **84% 的学生已经在使用 AI 工具。只有 18% 觉得自己具备了专业使用 AI 的能力。** 这门课程就是为了缩小这个差距。

这门课程来自开源项目 [AI Engineering from Scratch](https://github.com/rohitg00/ai-engineering-from-scratch)（MIT License），由我们整理为中文版本，方便中文学习者系统化学习。

### 课程特点

- **473 课时**，覆盖从线性代数到多 Agent 系统的完整知识体系
- **4 种编程语言**：Python、TypeScript、Rust、Julia
- **每课都有产出物**：一个 Prompt、一个 Skill、一个 Agent 或一个 MCP Server
- **先手写，再用框架**：先从原始数学推导实现，再用 PyTorch 等框架验证
- **免费开源**，MIT License
